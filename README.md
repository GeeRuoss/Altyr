# Altyr

Site vitrine one-page pour **Olivier Genet** : consulting en cabanes alpines (conseil + direction de travaux pour la conception, la rénovation et l'exploitation des refuges de montagne).

## Stack

HTML/CSS/JS vanilla, un seul fichier (`index.html`). Aucun build, aucune dépendance hors Google Fonts. Destiné à GitHub Pages.

## Direction visuelle

Dark alpin, froid, un seul accent (glacier). Toute la palette est dans `:root` (en haut de `index.html`). Signature graphique : courbes de niveau animées en canvas (`buildTopo()`).

- Titres : Bodoni Moda (serif)
- Corps / nav : Hanken Grotesk
- Données / altitudes : IBM Plex Mono
